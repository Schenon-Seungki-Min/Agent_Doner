> **작성일:** 2026.03.13 (Doner v5, Session #13)
> **목적:** Coree의 38~50세 인생 설계 통합 문서
> **상태:** v1 (초안 확정, 반복 업데이트 예정)

## 1. 북극성 (North Star)

### 핵심 체인

```
Employability(시장 가치) 유지 → 커리어 옵션 보장 → 경제적 안전망 자동 확보 → 가족 시간 확보
```

### 50세의 하루

오너 직속 팀장/임원으로 출근 → 7시 퇴근 → 중학생 딸+아내와 대화 → 개인 회사 업무 처리 → 리트리버와 밤산책 → 취침. 주말은 개인 회사+운동. **현재 라이프스타일의 업그레이드 버전.**

### 절대 포기 불가

- 커리어 성장 + 가족 시간 (동시 추구)
- 경제적 자유는 Employability가 보장하는 구조

### 절대 피할 미래

- "쓸모없어져서 자영업/인력사무소를 떠도는 50대"
- **핵심 공포:** 시장에서의 가치 소멸

### 설계 원칙 3가지

1. **"개인 회사"가 50세에 존재해야 한다** — KKAM 시리즈가 씨앗
2. **Employability > Title** — 직함보다 "갈 곳이 항상 있는 사람", 커리어 옵션 2개+ 유지
3. **라이프스타일은 현재 유지** — 혁명적 전환 없이 점진적 업그레이드

### 모든 의사결정의 첫 번째 필터

> "이걸 하면 내 시장 가치가 올라가나?"

## 2. 도메인별 현황 진단

### 도메인 A: 커리어

| 항목 | 현재 (38세) | 목표 (50세) | 갭 |
|------|------------|------------|-----|
| 포지션 | DHC 기획조정실 PM (실무자) | 오너 직속 팀장/임원 | 2단계 점프 필요 |
| 시장 가치 | DTx 상용화 PM (한국 유일) + BD/CD 경험 | 헬스케어 사업개발 임원급 | 매출 실적 부족 |
| 옵션 수 | 4개 (DHC/로킷/VC/녹십자) | 항상 2개+ | 현재 OK, 유지가 관건 |
| 리스크 | SPM 정치, 낙하산 위 상사 | - | 이방원 플레이로 관리 중 |

**커리어 옵션 4개:**

- **A) DHC 잔류:** 이방원 플레이, 실장 라인 킹메이커, 2027 Go/No-Go
- **B) 로킷:** 1.4억, 총괄사장=친구, 타이밍 자유 (5/25 데드라인)
- **C) VC 바이오그룹:** 총괄 바로 아래 복귀
- **D) 녹십자홀딩스:** 지주사 전략2팀, 사업기획/BD, AI Native 필수

### 도메인 B: 가정

| 항목 | 현재 | 목표 (50세) | 갭 |
|------|------|------------|-----|
| 자녀 | 딸 27개월, 어린이집 적응 | 중학생 딸과 매일 대화 | 시간 확보가 관건 |
| 배우자 | 커리어 중단 불안 + 우울 증상 케어 중 | 안정 + 사이드 협업 파트너 | 예측 불가 변수 |
| 반려동물 | 없음 | 리트리버 1마리 | 주거 안정 후 |
| 가용 시간 | 평일 2h(에너지15), 주말 9h(에너지80) | 7시 퇴근 후 가족 시간 | 현재 심각하게 부족 |

**와이프 상태 구조:**

- **표면:** 약한 약 처방, 필요 시 복용
- **실체:** 폭발 시 욕+자해 공갈 → 간헐적이더라도 케어 필요 수준
- **근본 원인:** 커리어 중단이 핵심 (본인 자각 완료), 육아 번아웃+부부 갈등 복합
- **부부 공통 공포:** Employability 상실 (다른 방식으로 표출)

**갈등 트리거 3개:**

1. 가사/육아 분담 불균형 (구조적 — 평일 에너지 부족)
2. 소통 방식 (Coree의 예민함/직설적 성향)
3. 와이프의 오래된 우울 증상 (통제 밖 변수)

**가정 설계 원칙:**

- **P1:** 와이프에게 "같이 뭔가 하고 있다"는 감각 → KKAM_SLP 공동 프로젝트화
- **P2:** 평일 에너지 15를 "보이게" 쓰기 → 양보다 질, 가시적 행동
- **P3:** 와이프 우울 증상은 통제 불가 변수 → 악화 방지 환경 설계
- **P4:** 소통 방식은 장기 개선 과제

## 3. 도메인 간 상충/시너지

### 상충

| 조합 | 내용 |
|------|------|
| 커리어 × 가정 | 이방원 플레이 에너지 소모 → 가정 에너지 감소 |
| 사이드 × 가정 | 프로젝트 작업 = "또 일해?" 갈등 트리거 |
| 자산 × 커리어 | 부동산 매수 → 대출 부담 → 이직 유연성 저하 |
| 커리어 × 사이드 | 이직 시 적응기 6개월+ → 사이드 올스톱 |

### 시너지

| 조합 | 내용 |
|------|------|
| 가정 × 사이드 | **KKAM_SLP = 와이프 참여 → 가정+사이드 동시 해결 (핵심 시너지)** |
| 커리어 × 사이드 | 포트폴리오 챗봇, KKAM_BIZ = 이직 무기 |
| 커리어 × 자산 | 연봉 상승 → 대출 여력 → 부동산 |

## 5. 리소스 배분 원칙

### 현실 제약

- 평일 가용 시간: ~1시간, 에너지 15/100
- 주말: 9시간이지만 가정에 에너지 80 투입
- **70(현직):30(사이드) 원칙 유지**

### 배분 매트릭스

| 구분 | 평일 (1시간) | 주말 (가정 외 여유분) |
|------|-------------|---------------------|
| 현직 연장 | 30분 (SleepQ 전략, 이방원 플레이) | 필요 시만 |
| KKAM_SLP | 15분 (와이프와 함께 = 가정 시간 겸용) | 주말 메인 — 와이프와 같이 |
| KKAM_MLP | 15분 (jeong 커뮤니케이션, 리뷰) | 필요 시만 |
| 커리어 관리 | 이직 시장 모니터링은 Doner 서포트 | 면접 준비 등 필요 시 |

### 핵심 설계 트릭

> KKAM_SLP을 "사이드 프로젝트 시간"이 아니라 **"와이프와 함께하는 시간"**으로 리프레이밍
> → 가정 에너지와 사이드 에너지가 중복 카운트 → 갈등 트리거 "또 일해?" 방지
