### 시뮬레이션 완료 — 핵심 답변 확정본

**자기소개 (2분):** 8년 디지털헬스케어 → 매번 스케일업 → SleepQ BD 10개사+예산 11배+인력 13배 → 2020년부터 GC 벤치마크 → 지주사 레벨이 다음 단계

**이직 사유:** SleepQ Incubation 완수 → ETC 조직으로 커머셜 핸드오프 중(종병 리스팅은 사내 유일 역량으로 서포트 지속) → 웰트 IPO 지원 + 닥터다이어리 BD로 역할 확장 시점에 제안 → 커리어가 리듬 타고 올라온 자연스러운 다음 스텝

**왜 GC인가 (압살 답변):** 대웅·동아·종근당은 처방 제품 중심 확장. GC만 유일하게 데이터 생태계를 먼저 구축하고 서비스를 전개하는 큰 판을 그렸다. GC메디아이 전환이 증거. 업계 리드는 시간 문제 → 그 판을 짜는 데 참여하고 싶다.

**BD 실패 경험 (마운자로):** 업계 네트워크로 릴리 부사장 인텔리전스 최초 입수 → 디지털 전략 전환 직접 발제 → 회장까지 IR 전략 직접 메일 → 비대면진료 2사+삐약 엮어 Co-pro 역제안 → 최종 라운드 진출 (릴리 단독판매 결정으로 종료). 핵심: 네트워크→인텔리전스→발제→실행을 한 사람이 드라이브.

**API/기술 명세서:** EMR-SleepQ API 연동 기획, Connect DTx 미들웨어 key handler, 비대면처방+실손 API 시스템 기획, AI PoC 데이터 명세서 직접 작성, FDA 규제→기술 명세서 전환. + 현재 닥터다이어리 SaaS hub 기획 중(AICC+복수 치료제 연결 = GC메디아이 Medical OS와 동일 구조). Translator + 필요시 명세서 직접 작성 가능.

**AI Native:** 총괄 에이전트(Doner) 설계, GitHub 기반 장기 메모리, 프로덕트 3개 직접 구축 — 단 코딩은 AI가 수행, 본인은 PM. "저는 PM이고 AI가 개발자인 셈." 대화형 AI 포트폴리오(kkam-gc.vercel.app) 면접장에서 시연 가능.

**약점:** ①품고 가는 시간이 길어 핸드오프 타이밍이 늦을 수 있음 → SOP/시스템 만들어서 구조화로 해결 중. ②핵심에 집중하다 이해관계자 감정적 온도를 놓칠 때 → 사전 정지 작업 습관으로 개선 중.

**연봉:** 현재 total 7,150만. 면접에서는 "헤드헌터 통해 전달" → 캐물으면 "8,000 이상, 유연하게 협의" → 더 캐물으면 "세부는 헤드헌터 조율"로 3단 방어.

**입사 후 플랜:** 6개월 = 가족사 완전 숙지 + 카운터파트 신뢰 구축 + 즉시 적용 가능 영역 병행 기여. 12개월 = 계열사 이해관계 정렬, 지주사 컨트롤타워로서 align 드라이브.
