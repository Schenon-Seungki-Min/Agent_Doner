02 가족사별 사업 분석

GC메디아이 (구 유비케어)

★★★ 핵심

현재 EMR 점유율 1위 (45-50%, 3만여 기관). '25 영업이익 75억 (+46%)

전환 3/24 주총 사명변경 확정. 의사랑 AI 고도화 / 클라우드 EMR / 데이터 의료서비스 확장

"Medical OS" = 병·의원·약국 + 제약사·보험·금융·헬스케어 기업 연결하는 데이터 생태계

  • 의사랑 AI: STT → SOAP 노트 자동 → 보험 청구 코드 추천 → 운영 자동화
  • 자체 LLM (의료 행정 + 제약/경영 분석 데이터)
  • KIMES 2026에서 상용화 공개 → 신규 계약 20%↑
  • 베트남 하노이 AI 정밀검진 센터 (HIS/LIS/PACS/EMR 통합)
  • 김진태 대표 = GC케어·GC메디아이 겸임 (맥킨지·현대카드)

BD 2팀 Medical OS → 사업 모델화하려면 외부 파트너 필요 = BD 2팀 업무. 내 SleepQ BD가 프로토타입

GC케어

★★ 중간지주

현재 기업 건강검진 대행 1위. 500여 센터, 250만 고객. 어떠케어(AI 검진 리포트, 9,000 센터 비교예약)

이슈 4년 연속 적자. PE 풋옵션 720억 회수. IPO 무산

BD 2팀 250만 데이터 + GC메디아이 EMR = 그룹 데이터 시너지 핵심 고리. 내 테서 BD와 매핑

GC녹십자

★ 캐시카우

혈액제제(알리글로) + 백신 + 희귀질환. 알리글로 미국 1,500억. '25 3Q 최초 분기 6,000억 (+31%)

글로벌: 미국(큐레보 시리즈B 1,600억) / 중국(CR) / 러시아 / 페루 / 베트남. mRNA 백신 임상 1상

BD 2팀 "녹십자 파이프라인 × GC메디아이 데이터 = RWE 기반 신약 포지셔닝" 시너지용

GC지놈

코스닥 '25.06

아이캔서치: AI 다중 암 조기 선별(액체생검). 6종 → 10종 확대('26). 글로벌 진출 계획

이슈 매출 80% 녹십자의료재단 내부거래 → 외부 다변화 과제

BD 2팀 GC메디아이 EMR × GC지놈 유전체 = 정밀의료 사업 모델

비브로스 (똑닥)

회원 680만+

병원 접수예약 O2O 1위. EMR 14개 연동. 접수→결제→실손 간편청구 원스톱

녹십자홀딩스 직접 주주 참여 (시리즈D 브릿지) → B2C 의지. 창업자 퇴임 후 새 대표 체제

BD 2팀 환자 접점(Patient Touchpoint) = Medical OS의 B2C 엔트리포인트

BD 2팀 "선택과 집중" — Non-core 정리 시그널. 면접에서 활용 가능

기타 가족사 (한줄 요약)

04 면접용 시너지 스토리

스토리 1 — "GC메디아이가 하려는 것을 이미 해봤다"

SleepQ: 에이치디정션(EMR) + 테서(검진) + 닥터나우/나만의닥터(비대면) + 실손 자동발급 = GC메디아이 Medical OS가 연결하려는 생태계의 프로토타입

스토리 2 — "5년간 안 됐던 시너지, 구조를 설계할 수 있다"

VC 바이오 4~5개 PMI + Steering Committee 이해관계 정렬. GC케어-GC메디아이-비브로스 시너지 → KPI 정렬이 첫 번째 할 일

스토리 3 — "아토머스는 이미 나의 파트너"

GC 투자 포트폴리오 기업 아토머스(마인드카페). SleepQ에서 직접 협업 — 대기업 EAP 1/6 불면증 → 처방 flow 설계. 합류 시 그룹 확대
05 함정 질문 대비
💣 "GC케어 적자인 거 아시죠?"
네. 다만 500여 센터 + 250만 고객 인프라가 있고, 이 위에 데이터 사업을 얹는 것이 핵심. 적자 원인이 사업 모델이 아니라 시너지 실행 속도라면, 가속하는 것이 제 영역입니다.
💣 "헥톤프로젝트 매각은 어떻게 보십니까?"
GC메디아이 Medical OS 전환 시점에 리소스 집중하는 합리적 판단. 선택과 집중의 시그널이고, 그 방향이 제가 하고 싶은 일과 일치합니다.
